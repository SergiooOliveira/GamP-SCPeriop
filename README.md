# GamP_SCPeriop

## Objetivo do Projeto
O GamP_SCPeriop é uma plataforma interativa focada na gestão, acompanhamento e gamificação de percursos de aprendizagem, desenhada para o contexto de supervisão (como o ensino perioperatório). O objetivo principal é facilitar a vida aos supervisores na estruturação de planos de estudo e, em simultâneo, aumentar a motivação e o foco dos estudantes através de um sistema de progressão recompensador.

## O Que o Projeto Faz Atualmente
* **Estruturação de Ensino:** Permite aos supervisores criar e gerir percursos de aprendizagem detalhados (Pathways), divididos por módulos, componentes teóricos/práticos e fases temporais.
* **Acompanhamento de Estudantes:** Atribuição de planos de estudo a alunos e monitorização direta do seu progresso global.
* **Sistema de Moldes (Templates):** Utilização de templates base para clonar e agilizar a criação de novos percursos.
* **Gamificação Base:** Interface interativa de "Conquistas" (Badges) do lado do aluno, para visualização de medalhas ganhas consoante o seu progresso no percurso.

## Futuras Implementações (Roadmap)

**Gamificação e Experiência do Utilizador (UX)**
* Melhoria do estilo visual das badges (aumentar o contraste dos cartões com o fundo da página para maior destaque).
* Implementação de um sistema de notificações para alertar o aluno no momento exato em que recebe uma nova badge.
* Introdução de um sistema de *Tiers* (níveis) para as badges (ex: 5 níveis distintos de conquista).

**Motor de Regras e Lógica de Badges**
* Criação de uma interface prática para adicionar novas badges ao sistema.
* Desenvolvimento do motor de *triggers*, permitindo definir exatamente que ações ou percentagens desbloqueiam cada badge.

**Gestão de Templates e Avaliação**
* Inclusão de elementos obrigatórios para avaliação nos *Pathway Templates*.
* Introdução de sistema de pesos (weights) estruturados dentro dos templates.

**Administração e Controlo de Acessos**
* Criação de um Dashboard exclusivo para o Administrador, que servirá como central de controlo para a criação global de Badges e *Pathway Templates*.
* Remoção da opção "Adicionar parâmetros" do painel do Supervisor, transferindo e centralizando essa responsabilidade no Administrador.
